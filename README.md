React Notes App

Простое приложение для заметок на React с автосохранением в localStorage.

Быстрый старт
npm install
npm start

Функции

* Создание, редактирование, удаление заметок
* Автосохранение в localStorage
* Первая заметка при первом запуске
* Адаптивный дизайн
* Поиск по заметкам

Технологии

* React 18
* JavaScript (ES6+)
* CSS3
* localStorage
